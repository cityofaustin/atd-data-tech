# Document Storage

## DTS Google Drive

#### **​**[**TPW DTS Google Drive**](https://drive.google.com/drive/folders/1fNmU-czryk5wJsn1gmb4WYUJJdut8Me7)

We love Google Drive, Docs, Sheets, and Slides for collaborative document editing and management. Since CTM does not have an enterprise Google subscription, we follow [these account set-up procedures](creating-a-google-account-as-a-city-employee.md) when we create our @austintexas.gov Google account.&#x20;

All of our **Google documents must be stored in the** [**TPW DTS folder**](https://drive.google.com/drive/u/0/folders/1fNmU-czryk5wJsn1gmb4WYUJJdut8Me7) owned by our shared resource account, coaatdamd@austintexas.gov. This allows everyone to find and view all files, even if the owner is out of office or leaves the team. It also ensures that we can comply with the City's record retention policies and fulfill Public Information Requests (PIRs) if necessary.&#x20;

{% hint style="danger" %}
The City's Information Security Office approved our use of Google **contingent upon our ability to comply with the** [**Texas Public Information Act**](https://www.texasattorneygeneral.gov/open-government/members-public/overview-public-information-act). Opening a Gmail account or storing files outside our shared folder may lead to a retraction of that approval.&#x20;
{% endhint %}

In the case that we need to store [CJIS](https://www.fbi.gov/services/cjis/cjis-security-policy-resource-center) and [HIPAA](https://www.hhs.gov/hipaa/for-professionals/security/laws-regulations/index.html) data, along with any personally identifiable information about individuals in the community, we need to arrange secure storage. **Under no circumstances should this sensitive information be stored in Google Drive.**

## Special File Storage Options

*   Personal files—such as HR documents—can be stored on your&#x20;

    Windows C Drive (backed up by ATS) or [your OneDrive](https://cityofaustin-my.sharepoint.com/).
* Files from Windows-only applications—like ArcGIS or Visio—are stored on the [G Drive](./#g-drive).
* Docs shared with City staff outside our division are stored in [TPW DTS SharePoint](https://cityofaustin.sharepoint.com/sites/tpwDTS/Shared%20Documents/Forms/AllItems.aspx).&#x20;
* Docs shared with external users can be shared on an [External Collaboration Portal](https://cityofaustin.sharepoint.com/sites/M365/SitePages/External-Collaboration-Portal.aspx).
